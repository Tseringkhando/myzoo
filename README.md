# 🦁 MyZoo — Claybrook Zoo Management System
** Live: https://clayzoo.lovestoblog.com
**Full-Stack Software Engineering Project (Year 3)**  
Built independently using **PHP**, **MySQL**, **HTML**, **CSS**, and **JavaScript**

---

## 🌍 Overview

**MyZoo** is a full-stack zoo management system created for **Claybrook Zoo**.  
The project includes **two main modules**:

- 🧑‍💼 **Admin Backend** — for zoo management (animals, exhibits, events, staff, etc.)
- 👩‍👩‍👧‍👦 **Visitor Frontend** — for guests to explore zoo details, exhibits, and upcoming events

## 🛠️ Technologies Used

| Layer | Technology | Purpose |
|-------|-------------|---------|
| **Frontend** | HTML5, CSS3, JavaScript | User interface for visitors and admins |
| **Backend** | PHP | Core application logic and database interaction |
| **Database** | MySQL | Data storage for animals, staff, and events |
| **Server** | Apache (via XAMPP / WAMP / MAMP) | Local hosting and deployment |
| **Version Control** | Git & GitHub | Source code management |

---

## ✨ Features

### 👩‍💼 **Admin Panel**
- Secure login for administrators  
- Add, edit, and delete **animal records**, **staff**, **events**, and **exhibits**  
- Manage **visitor inquiries** and **holiday schedules**  
- Dashboard overview for quick access  
- Validation and error handling for clean data management

### 🧭 **Visitor Frontend**
- Browse **animals by category or exhibit**
- View **upcoming events and holidays**
- Interactive and easy-to-navigate pages
- Responsive layout for a user-friendly experience
- Optional inquiry or feedback forms
